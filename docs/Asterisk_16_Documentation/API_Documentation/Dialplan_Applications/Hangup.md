# Hangup()

### Synopsis

Hang up the calling channel.

### Description

This application will hang up the calling channel.<br>


### Syntax


```

Hangup([causecode])
```
##### Arguments


* `causecode` - If a _causecode_ is given the channel's hangup cause will be set to the given value.<br>
