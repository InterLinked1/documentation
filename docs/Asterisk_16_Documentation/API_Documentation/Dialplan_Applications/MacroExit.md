# MacroExit()

### Synopsis

Exit from Macro.

### Description

Causes the currently running macro to exit as if it had ended normally by running out of priorities to execute. If used outside a macro, will likely cause unexpected behavior.<br>


### Syntax


```

MacroExit()
```
