# ODBC_Commit()

### Synopsis

Commits a currently open database transaction.

### Description

Commits the database transaction specified by _transaction ID_ or the current active transaction, if not specified.<br>


### Syntax


```

ODBC_Commit([transaction ID])
```
##### Arguments


* `transaction ID`
