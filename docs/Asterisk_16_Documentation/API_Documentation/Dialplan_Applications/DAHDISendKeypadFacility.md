# DAHDISendKeypadFacility()

### Synopsis

Send digits out of band over a PRI.

### Description

This application will send the given string of digits in a Keypad Facility IE over the current channel.<br>


### Syntax


```

DAHDISendKeypadFacility(digits)
```
##### Arguments


* `digits`
