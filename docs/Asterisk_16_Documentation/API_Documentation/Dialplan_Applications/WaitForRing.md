# WaitForRing()

### Synopsis

Wait for Ring Application.

### Description

Returns '0' after waiting at least _timeout_ seconds, and only after the next ring has completed. Returns '0' on success or '-1' on hangup.<br>


### Syntax


```

WaitForRing(timeout)
```
##### Arguments


* `timeout`
