# SpeechLoadGrammar()

### Synopsis

Load a grammar.

### Description

Load a grammar only on the channel, not globally.<br>

Hangs up the channel on failure. If this is not desired, use TryExec.<br>


### Syntax


```

SpeechLoadGrammar(grammar_name,path)
```
##### Arguments


* `grammar_name`

* `path`
