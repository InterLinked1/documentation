# Wait()

### Synopsis

Waits for some time.

### Description

This application waits for a specified number of _seconds_.<br>


### Syntax


```

Wait(seconds)
```
##### Arguments


* `seconds` - Can be passed with fractions of a second. For example, '1.5' will ask the application to wait for 1.5 seconds.<br>
