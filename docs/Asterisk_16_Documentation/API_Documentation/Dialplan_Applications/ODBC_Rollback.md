# ODBC_Rollback()

### Synopsis

Rollback a currently open database transaction.

### Description

Rolls back the database transaction specified by _transaction ID_ or the current active transaction, if not specified.<br>


### Syntax


```

ODBC_Rollback([transaction ID])
```
##### Arguments


* `transaction ID`
