# ADSIProg()

### Synopsis

Load Asterisk ADSI Scripts into phone

### Description

This application programs an ADSI Phone with the given script<br>


### Syntax


```

ADSIProg([script])
```
##### Arguments


* `script` - adsi script to use. If not given uses the default script *asterisk.adsi*<br>
