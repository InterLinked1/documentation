# AgentLogoff

### Synopsis

Raised when an Agent has logged off.

### Syntax


```


    Event: AgentLogoff
    Agent: <value>
    Logintime: <value>

```
##### Arguments


* `Agent` - Agent ID of the agent.<br>

* `Logintime` - The number of seconds the agent was logged in.<br>
