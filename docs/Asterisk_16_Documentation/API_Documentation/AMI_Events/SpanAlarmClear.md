# SpanAlarmClear

### Synopsis

Raised when an alarm is cleared on a DAHDI span.

### Syntax


```


    Event: SpanAlarmClear
    Span: <value>

```
##### Arguments


* `Span` - The span on which the alarm was cleared.<br>
