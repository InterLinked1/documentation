# FAXSessionsComplete

### Synopsis

Raised when all FAXSession events are completed for a FAXSessions command

### Syntax


```


    Event: FAXSessionsComplete
    [ActionID:] <value>
    Total: <value>

```
##### Arguments


* `ActionID`

* `Total` - Count of FAXSession events sent in response to FAXSessions action<br>
