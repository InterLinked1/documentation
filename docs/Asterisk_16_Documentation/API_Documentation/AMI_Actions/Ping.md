# Ping

### Synopsis

Keepalive command.

### Description

A 'Ping' action will elicit a 'Pong' response. Used to keep the manager connection open.<br>


### Syntax


```


    Action: Ping
    ActionID: <value>

```
##### Arguments


* `ActionID` - ActionID for this transaction. Will be returned.<br>
