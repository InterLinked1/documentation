# ParkedCalls

### Synopsis

List parked calls.

### Description

List parked calls.<br>


### Syntax


```


    Action: ParkedCalls
    ActionID: <value>
    ParkingLot: <value>

```
##### Arguments


* `ActionID` - ActionID for this transaction. Will be returned.<br>

* `ParkingLot` - If specified, only show parked calls from the parking lot with this name.<br>
