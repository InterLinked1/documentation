# QueueReset

### Synopsis

Reset queue statistics.

### Description

Reset the statistics for a queue.<br>


### Syntax


```


    Action: QueueReset
    ActionID: <value>
    Queue: <value>

```
##### Arguments


* `ActionID` - ActionID for this transaction. Will be returned.<br>

* `Queue` - The name of the queue on which to reset statistics.<br>
