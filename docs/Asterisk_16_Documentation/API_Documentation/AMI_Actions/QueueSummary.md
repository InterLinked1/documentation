# QueueSummary

### Synopsis

Show queue summary.

### Description

Request the manager to send a QueueSummary event.<br>


### Syntax


```


    Action: QueueSummary
    ActionID: <value>
    Queue: <value>

```
##### Arguments


* `ActionID` - ActionID for this transaction. Will be returned.<br>

* `Queue` - Queue for which the summary is requested.<br>
