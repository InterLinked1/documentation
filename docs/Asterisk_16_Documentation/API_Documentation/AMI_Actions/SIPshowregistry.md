# SIPshowregistry

### Synopsis

Show SIP registrations (text format).

### Description

Lists all registration requests and status. Registrations will follow as separate events followed by a final event called 'RegistrationsComplete'.<br>


### Syntax


```


    Action: SIPshowregistry
    ActionID: <value>

```
##### Arguments


* `ActionID` - ActionID for this transaction. Will be returned.<br>
