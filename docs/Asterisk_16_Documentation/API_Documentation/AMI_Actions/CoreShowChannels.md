# CoreShowChannels

### Synopsis

List currently active channels.

### Description

List currently defined channels and some information about them.<br>


### Syntax


```


    Action: CoreShowChannels
    ActionID: <value>

```
##### Arguments


* `ActionID` - ActionID for this transaction. Will be returned.<br>
