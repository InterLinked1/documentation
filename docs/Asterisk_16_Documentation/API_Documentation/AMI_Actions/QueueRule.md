# QueueRule

### Synopsis

Queue Rules.

### Description

List queue rules defined in queuerules.conf<br>


### Syntax


```


    Action: QueueRule
    ActionID: <value>
    Rule: <value>

```
##### Arguments


* `ActionID` - ActionID for this transaction. Will be returned.<br>

* `Rule` - The name of the rule in queuerules.conf whose contents to list.<br>
