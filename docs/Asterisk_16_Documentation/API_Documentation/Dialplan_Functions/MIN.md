# MIN()

### Synopsis

Returns the minimum of two numbers.

### Since

16.19.0, 18.5.0, 19.0.0

### Description

Returns the minimum of two numbers _num1_ and _num2_.<br>

``` title="Example: Sets the min variable equal to 4"

same => n,Set(min=${MIN(7,4)})


```

### Syntax


```

MIN(num1,num2)
```
##### Arguments


* `num1`

* `num2`
