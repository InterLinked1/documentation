# MAX()

### Synopsis

Returns the maximum of two numbers.

### Since

16.19.0, 18.5.0, 19.0.0

### Description

Returns the maximum of two numbers _num1_ and _num2_.<br>

``` title="Example: Sets the max variable equal to 13"

same => n,Set(max=${MAX(4,7)})


```

### Syntax


```

MAX(num1,num2)
```
##### Arguments


* `num1`

* `num2`
