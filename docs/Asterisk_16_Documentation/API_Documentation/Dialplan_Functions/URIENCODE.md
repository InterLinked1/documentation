# URIENCODE()

### Synopsis

Encodes a string to URI-safe encoding according to RFC 2396.

### Description

Returns the encoded string defined in _data_.<br>


### Syntax


```

URIENCODE(data)
```
##### Arguments


* `data` - Input string to be encoded.<br>
