# KEYPADHASH()

### Synopsis

Hash the letters in string into equivalent keypad numbers.

### Description

``` title="Example: Returns 537"

exten => s,1,Return(${KEYPADHASH(Les)})


```

### Syntax


```

KEYPADHASH(string)
```
