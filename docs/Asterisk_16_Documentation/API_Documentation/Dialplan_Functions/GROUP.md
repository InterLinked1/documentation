# GROUP()

### Synopsis

Gets or sets the channel group.

### Description

category can be employed for more fine grained group management. Each channel can only be member of exactly one group per category.<br>


### Syntax


```

GROUP(category)
```
##### Arguments


* `category` - Category name.<br>
