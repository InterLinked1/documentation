# IAXVAR()

### Synopsis

Sets or retrieves a remote variable.

### Description

Gets or sets a variable that is sent to a remote IAX2 peer during call setup.<br>


### Syntax


```

IAXVAR(varname)
```
##### Arguments


* `varname`
