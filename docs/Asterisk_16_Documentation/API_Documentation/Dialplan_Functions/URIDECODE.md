# URIDECODE()

### Synopsis

Decodes a URI-encoded string according to RFC 2396.

### Description

Returns the decoded URI-encoded _data_ string.<br>


### Syntax


```

URIDECODE(data)
```
##### Arguments


* `data` - Input string to be decoded.<br>
