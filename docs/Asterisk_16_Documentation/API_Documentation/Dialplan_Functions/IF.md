# IF()

### Synopsis

Check for an expression.

### Description

Returns the data following '?' if true, else the data following ':'<br>


### Syntax


```

IF(expression?retvalue)
```
##### Arguments


* `expression`

* `retvalue`

    * `true`

    * `false`
