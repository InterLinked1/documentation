# QUOTE()

### Synopsis

Quotes a given string, escaping embedded quotes as necessary

### Description

Example: $\{QUOTE(ab"c"de)\} will return ""ab\"c\"de""<br>


### Syntax


```

QUOTE(string)
```
##### Arguments


* `string`
