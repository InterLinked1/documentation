# QUEUE_EXISTS()

### Synopsis

Check if a named queue exists on this server

### Description

Returns 1 if the specified queue exists, 0 if it does not<br>


### Syntax


```

QUEUE_EXISTS(queuename)
```
##### Arguments


* `queuename`
