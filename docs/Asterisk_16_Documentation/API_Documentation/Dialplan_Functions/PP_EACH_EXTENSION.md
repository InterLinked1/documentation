# PP_EACH_EXTENSION()

### Synopsis

Execute specified template for each extension.

### Description

Output the specified template for each extension associated with the specified MAC address.<br>


### Syntax


```

PP_EACH_EXTENSION(mac,template_file)
```
##### Arguments


* `mac`

* `template_file`
