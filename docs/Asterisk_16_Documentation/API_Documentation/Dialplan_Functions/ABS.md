# ABS()

### Synopsis

Returns absolute value of a number.

### Since

16.19.0, 18.5.0, 19.0.0

### Description

Returns the absolute value of a number _num_.<br>

``` title="Example: Sets the absval variable equal to 13"

same => n,Set(absval=${ABS(-13)})


```

### Syntax


```

ABS(num)
```
##### Arguments


* `num`
