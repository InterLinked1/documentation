# DIALPLAN_EXISTS()

### Synopsis

Checks the existence of a dialplan target.

### Description

This function returns '1' if the target exits. Otherwise, it returns '0'.<br>


### Syntax


```

DIALPLAN_EXISTS(context,extension,priority)
```
##### Arguments


* `context`

* `extension`

* `priority`
