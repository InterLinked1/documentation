# SRVQUERY()

### Synopsis

Initiate an SRV query.

### Description

This will do an SRV lookup of the given service.<br>


### Syntax


```

SRVQUERY(service)
```
##### Arguments


* `service` - The service for which to look up SRV records. An example would be something like '\_sip.\_udp.example.com'<br>
