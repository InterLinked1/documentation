# SPEECH LOAD GRAMMAR

### Synopsis

Loads a grammar.

### Description

Loads the specified grammar as the specified name.<br>


### Syntax


```

SPEECH LOAD GRAMMAR GRAMMAR NAME PATH TO GRAMMAR 
```
##### Arguments


* `grammar name`

* `path to grammar`
