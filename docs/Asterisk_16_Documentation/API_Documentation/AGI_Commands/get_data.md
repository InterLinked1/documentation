# GET DATA

### Synopsis

Prompts for DTMF on a channel

### Description

Stream the given _file_, and receive DTMF data.<br>

Returns the digits received from the channel at the other end.<br>


### Syntax


```

GET DATA FILE TIMEOUT MAXDIGITS 
```
##### Arguments


* `file`

* `timeout`

* `maxdigits`
