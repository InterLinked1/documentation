# DATABASE DELTREE

### Synopsis

Removes database keytree/value

### Description

Deletes a _family_ or specific _keytree_ within a _family_ in the Asterisk database.<br>

Returns '1' if successful, '0' otherwise.<br>


### Syntax


```

DATABASE DELTREE FAMILY KEYTREE 
```
##### Arguments


* `family`

* `keytree`
