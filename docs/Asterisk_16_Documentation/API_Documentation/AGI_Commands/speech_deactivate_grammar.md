# SPEECH DEACTIVATE GRAMMAR

### Synopsis

Deactivates a grammar.

### Description

Deactivates the specified grammar on the speech object.<br>


### Syntax


```

SPEECH DEACTIVATE GRAMMAR GRAMMAR NAME 
```
##### Arguments


* `grammar name`
