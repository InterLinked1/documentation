# SPEECH RECOGNIZE

### Synopsis

Recognizes speech.

### Description

Plays back given _prompt_ while listening for speech and dtmf.<br>


### Syntax


```

SPEECH RECOGNIZE PROMPT TIMEOUT OFFSET 
```
##### Arguments


* `prompt`

* `timeout`

* `offset`
